# AI-Threat-Hunting-Assistant
## Overview
Develop an AI-powered cybersecurity assistant capable of translating natural-language threat-hunting requests into Microsoft Sentinel KQL queries to improve analyst efficiency and strengthen cyber defense operations for critical infrastructure organizations.

## Objective
- Reduce threat-hunting query development time
- Improve SOC analyst productivity
- Standardize KQL query generation
- Support proactive threat detection workflows
- Enhance cyber resilience across critical infrastructure organizations
## Features
- Natural-language to KQL translation
- MITRE ATT&CK mapping
- Threat hunting recommendations
- Investigation workflow guidance
- Exportable hunting queries
## Technologies
- Python
- Streamlit
- OpenAI API
- Microsoft Sentinel
- Kusto Query Language (KQL)
## Future Enhancements
- Multi-SIEM support
- Automated alert triage
- Incident response playbooks
- Vulnerability correlation
